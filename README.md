# Portafolio Profesional - Henry Morales

Portafolio web personal desarrollado con HTML5 y CSS3. Muestra mi perfil profesional, proyectos, experiencia, trayectoria, metas y eventos técnicos.

## Características

- Diseño responsive (adaptable a móviles y tablets)
- HTML semántico y bien estructurado
- Paleta de colores dark tech con acentos neón (#00d4ff)
- Fuente personalizada (Audiowide) cargada localmente
- Menú hamburguesa dinámico con CSS puro (checkbox hack)
- Navegación fluida entre secciones con scroll suave
- Animaciones de entrada (fade-in) en las secciones
- Galería de eventos asistidos con efectos hover
- Timeline educativo/profesional
- Sección de estadísticas visuales
- Metas y objetivos profesionales
- Contenedores centrados y organizados
- Footer con iconos de redes sociales (SVG)

## Secciones del Portafolio

1. **Inicio/Presentación** - Foto, nombre y perfil aspiracional
2. **Sobre mí** - Descripción personal e intereses
3. **Experiencia y Conocimientos** - Tecnologías y áreas de conocimiento
4. **Proyectos** - Proyectos académicos y personales
5. **Trayectoria (Timeline)** - Historial educativo y profesional
6. **Servicios** - Áreas de conocimiento y habilidades
7. **Eventos Asistidos** - Galería de fotos de eventos técnicos
8. **Metas y Objetivos** - Aspiraciones profesionales
9. **Contacto** - Información de contacto y redes sociales

## Estructura del Proyecto

```
Portafolio_Henry_HTML/
├── index.html              # Página principal
├── trabajos-tecnicos.html  # Página de trabajos técnicos
├── css/
│   └── style.css          # Estilos principales (unificado)
├── img/                   # Imágenes del proyecto
└── fonts/                 # Fuente Audiowide local
```

## Tecnologías Utilizadas

- HTML5 (semántico)
- CSS3 (Flexbox, Grid, Media Queries, Animaciones)
- CSS puro para menú hamburguesa (checkbox hack)
- SVG para iconos de redes sociales
- Fuentes locales (@font-face)
