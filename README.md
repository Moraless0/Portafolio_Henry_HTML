# Portafolio Profesional - Henry Morales

Portafolio web personal desarrollado con HTML5 y CSS3. Muestra mis proyectos, habilidades técnicas y trabajos técnicos en diferentes áreas de la tecnología.

## Características

- Diseño responsive (adaptable a móviles y tablets)
- HTML semántico
- Paleta de colores dark tech con acentos neón
- Fuente personalizada (Audiowide)
- Navegación fluida entre secciones
- Galería de trabajos técnicos con efectos hover

## Estructura del Proyecto

```
Portafolio_Henry_HTML/
├── index.html              # Página principal
├── trabajos-tecnicos.html  # Página de trabajos técnicos
├── css/
│   ├── style.css          # Estilos principales
│   └── trabajos.css       # Estilos de trabajos técnicos
├── img/                   # Imágenes del proyecto
└── fonts/                 # Fuentes locales
```

## Tecnologías Utilizadas

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Media Queries (Responsive Design)
